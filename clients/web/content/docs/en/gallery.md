# Asset Management

All AI-generated works are automatically archived with multi-dimensional management support.

## Auto-Archiving

After generation, works are automatically saved to your gallery. Each record includes:

- Generated images (multiple sizes)
- Prompts and parameters used
- Template information used
- Generation time and model version

## Organizing Works

### Tag Management

Add custom tags to works for easy categorization and retrieval:

- Batch add/remove tags
- Tags support multiple languages
- Filter works by tags

### Group Management

Organize works into different groups (projects/themes):

- Create custom groups
- Works can belong to multiple groups
- Groups support nested hierarchy

## Search & Filter

Multiple filter dimensions supported:

| Dimension | Description |
|-----------|-------------|
| Keywords | Search prompt content |
| Time range | Filter by creation date |
| Template | Filter by template used |
| Tags | Filter by tags |
| Groups | Filter by groups |

## Export & Share

### Export Formats

- **PNG** — Lossless compression, ideal for print
- **JPG** — Lossy compression, smaller file size
- **WebP** — Efficient compression, web optimized

### Batch Export

Select multiple works and download as a package.

## Privacy

- All works are **private by default**, visible only to the author
- Works are not used for model training
- Enterprise edition supports data isolation
