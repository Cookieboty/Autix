# Credits & Plans

Amux Studio uses a credit-based billing system. Each generation operation consumes credits.

## How to Earn Credits

| Method | Credits | Notes |
|--------|---------|-------|
| Subscription | Per plan quota | Auto-distributed monthly |
| Manual top-up | Per amount | Contact support or top up in dashboard |
| Daily check-in | +10 credits | First login each day |
| Complete creation | +5 credits | Max 50 credits/day |
| Invite friends | +200 credits | Issued after friend registers |

## Credit Consumption

Different generation modes consume different amounts:

| Action | Credits | Notes |
|--------|---------|-------|
| Standard text-to-image | 5 credits | Single standard resolution |
| HD text-to-image | 10 credits | Single high resolution |
| Image-to-image | 8 credits | Based on reference image |
| Inpainting | 6 credits | Regenerate specified region |

## Credit Expiration

- Subscription credits: Valid for the current month, no rollover
- Top-up credits: Never expire
- Reward credits: Valid for 90 days after earning

## Invoicing & Payment

- WeChat Pay and Alipay supported
