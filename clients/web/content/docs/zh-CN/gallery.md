# 作品管理

所有 AI 生成的作品自动入库，支持多维度管理。

## 作品入库

生成完成后，作品自动保存到你的作品库。每个作品记录包含：

- 生成结果图片（多尺寸）
- 使用的提示词和参数
- 使用的模板信息
- 生成时间和模型版本

## 作品组织

### 标签管理

为作品添加自定义标签，方便分类检索：

- 支持批量添加/移除标签
- 标签支持中英文
- 可按标签筛选作品

### 分组管理

将作品归入不同分组（项目/主题）：

- 创建自定义分组
- 作品可属于多个分组
- 分组支持嵌套层级

## 搜索与筛选

支持多种筛选维度：

| 维度 | 说明 |
|------|------|
| 关键词 | 搜索提示词内容 |
| 时间范围 | 按创建时间筛选 |
| 模板 | 按使用的模板筛选 |
| 标签 | 按标签筛选 |
| 分组 | 按分组筛选 |

## 导出与分享

### 导出格式

- **PNG** — 无损压缩，适合打印
- **JPG** — 有损压缩，文件更小
- **WebP** — 高效压缩，网页优化

### 批量导出

选择多个作品后一键打包下载。

## 隐私说明

- 所有作品**默认私有**，仅作者可见
- 作品不会用于模型训练
- 企业版支持数据隔离
