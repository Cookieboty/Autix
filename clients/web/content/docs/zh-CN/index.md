# 快速开始

欢迎使用 **Amux Studio** —— 为内容创作者打造的持续生成平台。

## 平台简介

Amux Studio 将提示词管理、模板复用、AI 图片生成、作品入库与交付整合为一条高效工作流，帮助创作者稳定量产高质量视觉内容。

## 30 秒快速上手

### 1. 注册账号

访问 [注册页面](/register) 创建账号。注册后管理员将在 1 个工作日内完成审核。

### 2. 进入工作台

审核通过后，登录并进入 [工作台](/chat) 开始你的第一次对话。

### 3. 使用模板创作

前往 [模板市场](/templates)，选择合适的提示词模板，填写变量参数后一键生成。

### 4. 管理作品

所有生成结果自动入库到 [作品库](/chat)，支持标签、分组管理和多格式导出。

## 核心概念

| 概念 | 说明 |
|------|------|
| **提示词模板** | 预定义的 AI 生成提示词，支持变量替换，可公开发布或私有使用 |
| **积分** | Amux 站内服务兑换单位，每次生成消耗积分，可通过订阅、积分包或活动获取，不具备现金价值 |
| **作品库** | 自动归档所有生成结果，支持搜索、分组、标签管理 |
| **工作台** | AI 对话界面，支持多轮对话、多模型切换 |

## 系统要求

- 现代浏览器（Chrome 90+、Firefox 88+、Safari 15+、Edge 90+）
- 稳定的网络连接
