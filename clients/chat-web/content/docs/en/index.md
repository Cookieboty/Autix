# Getting Started

Welcome to **Amux Design** — a continuous generation platform built for content creators.

## About the Platform

Amux Design integrates prompt management, template reuse, AI image generation, asset archiving, and delivery into one efficient workflow, helping creators produce high-quality visual content at scale.

## Quick Start in 30 Seconds

### 1. Create an Account

Visit the [registration page](/register) to create your account. An administrator will review it within 1 business day.

### 2. Enter the Workspace

Once approved, log in and head to the [Workspace](/chat) to start your first conversation.

### 3. Create with Templates

Browse the [Template Market](/templates), pick a prompt template, fill in the variables, and generate with one click.

### 4. Manage Your Works

All generated results are automatically archived in your [Gallery](/chat), with support for tags, groups, and multi-format export.

## Core Concepts

| Concept | Description |
|---------|-------------|
| **Prompt Template** | Pre-defined AI generation prompts with variable substitution, publishable or private |
| **Credits** | Platform currency consumed per generation, obtainable via subscription or top-up |
| **Gallery** | Auto-archives all generation results with search, grouping, and tag management |
| **Workspace** | AI conversation interface supporting multi-turn dialogue and model switching |

## System Requirements

- Modern browser (Chrome 90+, Firefox 88+, Safari 15+, Edge 90+)
- Stable internet connection
