# Workflow Overview

Amux Design simplifies the creation process into four core steps, taking you from inspiration to finished product in seconds.

## Overall Flow

```
Input Inspiration → AI Generation → Refine & Iterate → Export & Deliver
```

## Step-by-Step

### 1. Input Inspiration

- **Text Description**: Enter natural language descriptions directly in the workspace
- **Template Selection**: Choose a prompt template from the market and fill in variables
- **Reference Upload**: Upload reference images for style transfer or inpainting

### 2. AI Generation

The system calls AI models to generate images based on your input. Multiple generation modes are supported:

| Mode | Description | Use Cases |
|------|-------------|-----------|
| Text-to-Image | Generate from text descriptions | Posters, covers, social media |
| Image-to-Image | Generate based on reference images | Style transfer, variations |
| Inpainting | Regenerate specific regions | Detail edits, element replacement |

### 3. Refine & Iterate

Not satisfied with the result? Continuously optimize through multi-turn dialogue:

- Adjust prompts for more precise requirements
- Modify variable parameters to fine-tune effects
- Use inpainting to refine details

### 4. Export & Deliver

Export satisfactory works in multiple formats and sizes:

- **PNG** — Lossless, ideal for print
- **JPG** — Compressed, suitable for web distribution
- **WebP** — Efficient compression, optimized for web

## Best Practices

1. **Be specific with prompts**: The more detailed, the closer the result to expectations
2. **Leverage templates**: Templates contain proven prompt structures, more efficient than starting from scratch
3. **Use multi-turn dialogue**: Gradual refinement is more efficient than pursuing perfection in one shot
4. **Archive promptly**: Save satisfactory works immediately for future reuse
