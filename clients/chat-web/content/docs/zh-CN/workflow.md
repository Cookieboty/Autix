# 工作流概述

Amux Design 将创作流程简化为四个核心步骤，帮助你从灵感到成品只需几十秒。

## 整体流程

```
输入灵感 → AI 生成 → 优化迭代 → 导出交付
```

## 步骤详解

### 1. 输入灵感

- **文字描述**：在工作台直接输入自然语言描述你想要的图片
- **模板套用**：从模板市场选择合适的提示词模板，填写变量参数
- **参考图上传**：上传参考图片，AI 将基于参考进行风格迁移或局部重绘

### 2. AI 生成

系统根据你的输入调用 AI 模型进行图片生成。支持多种生成模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 文生图 | 纯文字描述生成图片 | 海报、封面、社媒配图 |
| 图生图 | 基于参考图生成 | 风格迁移、变体生成 |
| 局部重绘 | 指定区域重新生成 | 修改细节、替换元素 |

### 3. 优化迭代

生成结果不满意？可以通过多轮对话持续优化：

- 调整提示词描述更精确的需求
- 修改变量参数微调效果
- 使用局部重绘精修细节

### 4. 导出交付

满意的作品可以一键导出，支持多种格式和尺寸：

- **PNG** — 无损高清，适合印刷
- **JPG** — 压缩适中，适合网络分发
- **WebP** — 高效压缩，适合网页使用

## 最佳实践

1. **提示词要具体**：描述越详细，生成结果越接近预期
2. **善用模板**：模板包含经过验证的提示词结构，比从零写更高效
3. **利用多轮对话**：逐步细化比一次追求完美更有效率
4. **及时入库**：满意的作品立即收藏，方便后续复用
