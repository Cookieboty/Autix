# 售后常见问题

## Q1: 如何申请退货？

A: 登录账户 → 我的订单 → 选择订单 → 申请退货 → 填写退货原因 → 提交申请。客服会在 24 小时内审核。

## Q2: 退货运费谁承担？

A: 
- 质量问题、发错货：卖家承担运费
- 七天无理由退货：买家承担运费
- 部分商品支持上门取件服务

## Q3: 退款多久到账？

A: 
- 支付宝/微信：1-3 个工作日
- 银行卡：3-7 个工作日
- 钱包余额：实时到账

## Q4: 商品有质量问题怎么办？

A: 
1. 拍照/录像保留证据
2. 联系客服说明情况
3. 客服确认后提供退货地址
4. 寄回商品（运费到付）
5. 收到商品后 3-5 个工作日退款

## Q5: 可以只退款不退货吗？

A: 以下情况支持仅退款：
- 商品未发货
- 商品丢失
- 商品严重损坏无法寄回
其他情况需要退货后才能退款。

## Q6: 退货后多久能收到退款？

A: 卖家收到退货商品并确认无误后，3-5 个工作日内退款。

## Q7: 部分退货可以吗？

A: 可以。如果订单包含多件商品，可以选择部分商品退货，按实际退货商品金额退款。

## Q8: 赠品需要一起退回吗？

A: 是的。如果订单包含赠品，退货时需要将赠品一并退回，否则会从退款金额中扣除赠品价值。

## Q9: 退货地址是什么？

A: 退货地址会在退货申请审核通过后，通过短信和站内信发送给您。不同商品可能有不同的退货地址。

## Q10: 退货被拒绝怎么办？

A: 如果退货申请被拒绝，您可以：
1. 查看拒绝原因
2. 联系在线客服申诉
3. 提供补充证据
4. 平台客服会重新审核
