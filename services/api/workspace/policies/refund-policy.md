# 退款政策

## 退款适用场景

1. **七天无理由退货**：签收后 7 天内，商品未使用且包装完好
2. **质量问题退款**：商品存在质量问题，支持退货退款
3. **发错货退款**：卖家发错商品，支持退货退款，运费由卖家承担
4. **未收到货退款**：超出预计到达时间仍未收到商品

## 退款时效

| 退款方式 | 预计到账时间 |
|---|---|
| 原路退回（支付宝/微信）| 1-3 个工作日 |
| 原路退回（银行卡）| 3-7 个工作日 |
| 退回钱包余额 | 实时到账 |

## 退款流程

1. 申请退款 → 2. 卖家审核（24小时内）→ 3. 退货寄回 → 4. 平台确认收货 → 5. 退款到账

## 不予退款的情况

- 超过退款时效
- 商品人为损坏
- 缺少原始包装/配件
- 已激活的软件/会员类商品
- 定制类商品
