# 退货政策

## 退货条件

1. **时间限制**：商品签收后 7 天内可申请退货
2. **商品状态**：商品及包装完好，不影响二次销售
3. **特殊商品**：以下商品不支持退货
   - 定制商品
   - 已激活的数码产品
   - 贴身衣物

## 退货流程

1. 联系客服申请退货
2. 提供订单号和退货原因
3. 客服审核通过后，提供退货地址
4. 寄回商品（运费由买家承担，质量问题除外）
5. 收到商品后 3-5 个工作日退款

## 退款方式

- 原路退回至支付账户
- 到账时间：3-7 个工作日

## 注意事项

- 退货时请保留快递单号
- 商品损坏或丢失，买家承担责任
- 质量问题由卖家承担运费
